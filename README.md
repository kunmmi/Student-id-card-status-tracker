# Student ID Card Tracking & Collection System

## Overview
This project is a backend system designed to manage and track the production, readiness, and collection of student ID cards. It models real-world administrative workflows used in academic institutions.

## Features
- Student record lookup
- ID card status tracking (pending, ready, collected)
- Administrative status updates
- Clear backend workflows reflecting real processes

## Tech Stack
- Python
- Django (backend framework)
- SQLite / relational database concepts

## Purpose
The system was built to demonstrate backend application design, data consistency, and workflow-driven logic for institutional use cases.

## Status
Core backend functionality implemented.  
Project was developed as part of an academic/institutional solution.
